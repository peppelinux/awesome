# Awesome projects

## Network
- [bettercap.org](https://www.bettercap.org/) - The Swiss Army knife for WiFi, Bluetooth Low Energy, wireless HID hijacking and IPv4 and IPv6 networks reconnaissance and MITM attacks.

## Docker
- [portainer](https://github.com/portainer/portainer) - lightweight ‘universal’ management GUI that can be used to easily manage Docker, Swarm, Kubernetes and ACI environments.


## Health checks
- [Healthchecks](https://github.com/healthchecks/healthchecks) - Healthchecks is a cron job monitoring service. It listens for HTTP requests and email messages ("pings") from your cron jobs and scheduled tasks ("checks"). When a ping does not arrive on time, Healthchecks sends out alerts. Also available in Cloud, [here](https://healthchecks.io/).
